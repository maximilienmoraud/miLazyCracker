# miLazyCracker
Mifare Classic Plus - Hardnested Attack Implementation for LibNFC USB readers (SCL3711, ASK LoGO, etc)

Installation:
```bash
./miLazyCrackerFreshInstall.sh
```

Usage example: place a tag and enjoy
```bash
mkdir mydumps
cd mydumps
miLazyCracker
```
